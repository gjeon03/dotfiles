지정된 파일 또는 디렉토리에서 불필요한 코드를 찾아 정리한다.

## 검사 항목

1. **미사용 import**: 참조되지 않는 import 문
2. **미사용 변수/함수**: 선언되었지만 사용되지 않는 코드
3. **주석 처리된 코드**: 삭제해야 할 주석 코드 블록
4. **데드 코드**: 도달 불가능한 코드 경로
5. **중복 코드**: 명백한 복사-붙여넣기 패턴

## 절차

1. 대상 스캔 및 문제 목록 작성
2. 각 항목에 대해 위치와 제거 사유 표시
3. 제거해도 안전한 이유 설명
4. 확인 후 적용

## 규칙

- 동작 변경 없이 정리만 수행
- 의심스러운 경우 제거하지 않고 보고만
- 타입체크와 테스트가 깨지지 않는지 확인

대상: $ARGUMENTS
