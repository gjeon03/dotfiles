이번 세션을 분석하고 CLAUDE.md 개선점을 제안한다.

## 분석 항목

1. **반복된 실수**: 같은 지시를 여러 번 받은 경우
2. **누락된 컨텍스트**: 미리 알았다면 실수를 방지할 수 있었던 정보
3. **워크플로우 마찰**: 불필요하게 느리거나 수동이었던 단계
4. **발견된 패턴**: 세션 중 표현된 선호나 컨벤션

## 출력 형식

CLAUDE.md에 추가할 구체적인 내용을 제안한다:

```markdown
## 추가 제안

### [섹션명]
- [추가할 규칙이나 컨텍스트]
```

프로젝트별 CLAUDE.md와 글로벌 CLAUDE.md 중 어디에 추가해야 하는지도 구분한다.
